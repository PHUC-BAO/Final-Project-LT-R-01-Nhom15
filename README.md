# Final-Project-LT-R-01-Nhom15
Analyzing freelancer income across different industries

## Run analysis

Prepare a CSV file with at least these columns:
- `industry`
- `income`

Then run:

```bash
Rscript analyze_freelancer_income.R <input_csv> [output_csv]
```

The script prints a per-industry summary (freelancer count, total/average/median/min/max income).  
If `output_csv` is provided, the summary is saved to that file.
